# GLITCH

## Description :

A top-down cyberpunk hack 'n slash/shooter.
Destroy Glitches to proceed.

## Controls :
WASD - Walk.
Shift + WASD - Sprint
Middle Mouse Button - Dash.
LMB - Sword Attack / Laser Fire.
RMB - Energy Shield.
Scroll Wheel - Change Weapons.
